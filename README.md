Pacote final da Home A7 One.

- Desktop mantém:
  “A Dani entende seu momento e direciona você ao especialista certo para o próximo passo.”
- Mobile mostra:
  “A Dani é o primeiro passo. Nossa equipe transforma estratégia em resultados.”

## Marca e compartilhamento
- Favicon oficial com somente o símbolo A7.
- Ícone para tela inicial do iPhone.
- Título correto da aba do navegador.
- Meta description para Google.
- Open Graph para WhatsApp, LinkedIn e Facebook.
- Imagem de compartilhamento própria da A7 One.

## Publicação
1. Descompacte o ZIP.
2. Envie somente o conteúdo interno para o mesmo repositório `a7one-assinatura`.
3. Substitua os arquivos existentes.
4. A Vercel publicará automaticamente.

O favicon e a imagem de compartilhamento podem levar alguns minutos para atualizar por causa do cache do navegador e do WhatsApp.
